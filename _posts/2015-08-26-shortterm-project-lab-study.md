## 프로그램 정보

* **이름** : 스터디 사이트
* **유형** : 학교 프로젝트 - JSP
* **기간** : 4주 (15.05.25 ~ 15.06.19)
* **담당** : 팀장
	* 게시판(스터디 등록, 홍보) 구현
	* 덧글(Q&A) 구현
	* 회원 관련 구현(로그인&로그오프, 회원가입, 비밀번호 찾기, 관리자 메뉴, 프로필, 프로필 수정)
	* 서적 추가(네이버 OpenAPI를 사용)
	* 스터디 신청 및 수락 구현

## 개발 환경

* 백 엔드
	* Mac OS
	* FreeMarker & Spring Boot
	* MySQL & JPA
	* Fake SMTP Server
	* Gradle
* 프론트 엔드
	* Javascript & jQuery
	* Bootstrap
* 협업 도구
	* github
	* waffle.io
* IDE
	* IntelliJ
