## 프로그램 정보

* **이름** : Po Tweeter
* **유형** : 스터디 프로젝트 - Java
* **기간** : 6주
	* 준비 : 4주
	* 코딩 : 2주
* **담당** : 팀장
	* 클라이언트 & 서버(GUI) 구현
	* 이모티콘 구현
	* 이미지 및 파일 전송 구현
	* 프로필 구현
	* 공지 구현

## 개발 환경

* Windows 7
* Linux (Centos)
* Java
* Eclipse + WindowBuilder
* Mysql
